# Referee Report Format

## Overview
Every match must have a referee report submitted by the assigned referee after the match is completed. The report captures the referee's observations and any notable incidents.

## Report Structure

### Match Information (Auto-populated)
- Match ID
- Match Date
- Kickoff Time
- Venue
- Home Team
- Away Team
- Final Score
- Referee Name

### Match Control Notes (Referee Input)

#### General Comment
Free-text field for the referee's overall assessment of the match. May include notes on the flow of play, key moments, or general impressions.

#### Time Management Observation
Notes on whether the match started on time, any delays, time added for stoppages, and whether both teams were punctual and ready.

#### Dress Code Observation
Notes on whether players were properly kitted (matching jerseys, shin guards, proper footwear). Flag any dress code violations.

#### Organization Observation
Notes on the overall organization of the matchday: were team sheets submitted on time, were lineups ready, was the pitch in acceptable condition.

#### Team Conduct Observation
Notes on the general behavior and sportsmanship of both teams. Covers interactions with the referee, opponents, and spectators.

#### Additional Incidents
Any other notable events not covered above. May include:
- Injuries requiring medical attention
- Crowd disturbances
- Abandoned or suspended play
- Equipment issues
- Pitch conditions affecting play

### Discipline Section (Linked from CardEvents)
For each card issued during the match:
- Player Name
- Team
- Card Type (Yellow / Red / Second Yellow)
- Minute
- Reason for Caution / Sending Off
- Additional Referee Note

## Submission Rules
1. A referee report can only be submitted for a match the referee is assigned to.
2. The report can be saved as a draft and submitted later.
3. Once submitted, only the referee or an admin can edit the report.
4. All discipline events (cards) are submitted separately but linked to the same match.

## Standard Caution Reasons (Reference)
| Code | Reason |
|------|--------|
| C1 | Unsporting behavior |
| C2 | Dissent by word or action |
| C3 | Persistent fouling |
| C4 | Delaying the restart of play |
| C5 | Failure to respect required distance |
| C6 | Entering/leaving without permission |
| C7 | Deliberate handball |

## Standard Sending Off Reasons (Reference)
| Code | Reason |
|------|--------|
| S1 | Serious foul play |
| S2 | Violent conduct |
| S3 | Spitting at an opponent or other person |
| S4 | Denying a goal or goal-scoring opportunity by handball |
| S5 | Denying a goal-scoring opportunity by foul |
| S6 | Offensive, insulting, or abusive language/gestures |
| S7 | Receiving a second caution |
