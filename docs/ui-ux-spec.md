# UI/UX Specification

## Goals
Create a system that is easy for admins, referees, and team managers to use during fast-paced matchdays.

## Primary Design Goals
- Fast data entry
- Mobile-friendly referee workflow
- Easy historical search
- Clear event tracking
- Simple export flow

## Main Navigation
- Dashboard
- Matches
- Teams
- Players
- Referees
- Reports
- Exports
- Settings

## Key Screens
- Dashboard
- Matches list
- Match details
- Team lineup form
- Referee report form
- Card entry panel
- Upload center
- Export center

## Responsive Rules
- Desktop uses sidebar and wide tables.
- Mobile uses stacked cards and step-by-step forms.
- Referee forms should be touch-friendly.

## Feedback Rules
- Every save action should show success or error feedback.
- Every empty table should explain what is missing.
