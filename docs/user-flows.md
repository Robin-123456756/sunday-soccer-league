# User Flows

## Admin Flow
1. Log in
2. Create teams, players, referees, venues, and matchdays
3. Create match fixture
4. Assign referee
5. Review lineups and uploaded team sheets
6. Review discipline and referee reports
7. Export league data

## Referee Flow
1. Log in
2. Open assigned match
3. Record card events and reasons
4. Add comments and observations
5. Submit report

## Team Manager Flow
1. Log in
2. Open scheduled match
3. Select starters and bench players
4. Upload team sheet file
5. Submit lineup
